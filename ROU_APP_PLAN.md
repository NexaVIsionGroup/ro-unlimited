# RO Unlimited — App Platform Plan
> Compiled: 2026-03-08

## Architecture
- PWA first (install to homescreen, push notifications, offline)
- Then native wrappers: iOS (App Store), Android (TWA/Play Store)
- Fixed mobile app shell — locked viewport, bottom tab bar, animated drawer
- Built on Next.js + Supabase + Sanity

## Bottom Tab Bar (4 tabs)
1. Dashboard (home) — stats, active jobs, today schedule, activity feed
2. Jobs (hard hat) — all projects, tap into details
3. Messages (chat) — client comms, crew chat, NexaVision support
4. Menu (grid) — animated app drawer slides up
